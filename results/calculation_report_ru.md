## 2. Схема расчетов

1. Сгенерированы `52` стартовые геометрии B12: `12` регулярных и `40` случайных.
2. Для двух мультиплетностей подготовлено и выполнено `104` xTB-оптимизации.
3. После ранжирования и удаления дублей выделено `3` уникальных низкоэнергетических кандидата.
4. Для `3` кандидатов выполнена переоптимизация методом `r2SCAN-3c`.
5. Для тех же `3` структур выполнен расчет `NumFreq`.
6. В финальный набор вошли `2` структуры без мнимых частот по строгому критерию `n_imag_strict = 0`.

## 3. Формула расчета относительных энергий

Относительные энергии для каждого этапа рассчитывались по формуле:

`dE_i = (E_i - E_min) * 2625.49962`,

где `E_i` и `E_min` выражены в Hartree, а коэффициент `2625.49962` переводит разность энергий в `кДж/моль`.

## 7. Вывод по расчетам

В исследованном наборе структур наименьшую энергию имеет `double_ring_6_6_d1.90_m1_r2scan`, которая может рассматриваться как глобальный минимум в пределах выполненного многостартового поиска. Структура `ring12_d1.70_m1_r2scan` также является устойчивым локальным минимумом, однако уступает лидеру на `632.10 кДж/моль`.

Триплетная структура `zigzag_chain12_d1.50_m3_r2scan` не проходит частотную проверку: для нее получено `6` мнимых частот, из них `5` ниже порога `-20.0 см^-1`, а минимальная частота составляет `-2022.99 см^-1`. Поэтому она исключается из финального набора устойчивых структур.
