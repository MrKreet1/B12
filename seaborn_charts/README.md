# Seaborn Charts

This folder contains static Seaborn charts built from the current project outputs.

Run:

```bash
python seaborn_charts/generate_charts.py
```

Generated files are written to:

```text
seaborn_charts/out/
```

Open this file first:

```text
seaborn_charts/out/index.html
```

Source tables:

- `results/xtb_ranked_all.csv`
- `results/xtb_ranked_unique.csv`
- `results/final_numfreq_report.csv`
- `results/final_good_no_imag.csv`
